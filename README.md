# Arkisanté Architecture — Website (HTML & CSS)

This project is an institutional website developed for **MD Arkisanté Architecture**, a firm specialized in architectural design for healthcare environments. The main objective of this website is to **promote the office**, strengthen its **visual identity**, and serve as a **project catalog for clients**.

The project was built entirely with **HTML and CSS**, focusing on clean structure, semantic markup, and a well-organized layout using **Flexbox** and **CSS Grid**.

---

## Project overview

The purpose of this website is to:

- Present the architecture office in a professional and cohesive manner  
- Showcase the team, services, and architectural expertise  
- Display a gallery of projects and past work  
- Provide prospective clients with clear, structured information  
- Function as a static portfolio website

All layout design, copywriting, structural decisions, and front-end development were created manually and from scratch.

---
